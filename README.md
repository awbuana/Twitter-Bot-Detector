# Twitter-Bot-Detector
Deteksi Bot pada Twitter menggunakan R

## Deskripsi
Mendeteksi apakah suatu user bot atau bukan

## Kontributor
1. Azmi
2. Maulana Wisnu
3. Agni Wira Buana

## Fitur yang digunakan
```
- tweet_per_day
- mention_per_tweet
- retweet_per_tweet
- link_per_tweet
- hashtag_per_tweet
- average_sentiment
- average_distance (kemiripan suatu tweet dengan tweet lainnya)
- jumlah following
- jumlah follower
```
